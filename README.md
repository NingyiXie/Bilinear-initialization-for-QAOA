# Bilinear-initialization-for-QAOA
This repository is implementation of the ["A Depth-Progressive Initialization Strategy for Quantum Approximate Optimization Algorithm"](https://arxiv.org/abs/2209.11348)  

## Requirements
* Qiskit 0.32.1
* Numpy 1.21.4
* qaoalib 0.1.6
* Scipy 1.7.3
* NetworkX 2.6.3
